/clone begin (x, y, and z)

end (x, y, and z)

destination (x, y, and z)

replace

masked

normal

force

move

filtered

tile_name

tile_data

The begin argument is the starting x, y, and z coordinates for the region of blocks to clone.

The end argument is the ending x, y, and z coordinates for the block on the opposite corner of the region to clone.

The destination argument is the x, y, and z coordinates for the destination region. This MUST be the bottom northwest corner of the region in order for it to work how it is intended.

The replace argument is optional. This argument allows for cloning all blocks in the region, including air blocks. NOTE: This is the default behavior if you don't specify any optional arguments.

The masked argument is optional. This argument will allow for only cloning blocks that aren't air blocks.

The normal argument is optional. This argument allows for cloning the blocks from the specified source region to the specified destination region. NOTE: The is also default behavior.

The force argument is optional. This argument will force the region specified to clone, even if the source region and destination region overlap.

The move argument is optional. This argument, when filtered is not applied, will move the blocks in the specified source region to the specified destination region. With filtered applied, blocks that were not cloned will remain the same.

The filtered argument is optional. This argument will only clone blocks that match the specified tile_name argument.

The tile_name argument is optional. This argument represent the specific block you want to clone.

The tile_data argument is optional. This argument represents the meta data of the specified tile_name argument.
